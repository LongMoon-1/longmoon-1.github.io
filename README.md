# 翻译工具

即输即译 · 双语对照 · 一键复制

---

## 使用方式

1. 选择源语言 / 目标语言
2. 输入文字，点击翻译（或 Ctrl+Enter）
3. 点击“复制结果”直接取走译文

---

## 两种模式

- **快速翻译**：只输出译文，无额外内容
- **详细释义**：附带词汇解释和语法说明（使用源语言）

---

## 快捷操作

| 操作 | 说明 |
|------|------|
| Ctrl+Enter | 快速翻译 |
| 点击复制按钮 | 译文一键复制到剪贴板 |

---

## 支持的语种

中文、英文、日文、韩文、法文、德文、俄文、西班牙文、阿拉伯文、葡萄牙文、意大利文、荷兰文、瑞典文、波兰文、土耳其文、泰文、越南文、印地文、希腊文、希伯来文

# Translation Tool

Type · Translate · Copy

---

## How to Use

1. Select source / target language
2. Enter text, click Translate (or Ctrl+Enter)
3. Click "Copy" to get the result

---

## Two Modes

- **Fast**：translation only
- **Detailed**：adds explanations and grammar notes (in source language)

---

## Shortcuts

| Action | Description |
|--------|-------------|
| Ctrl+Enter | Quick translate |
| Copy button | One‑click copy |

---

## Supported Languages

Chinese, English, Japanese, Korean, French, German, Russian, Spanish, Arabic, Portuguese, Italian, Dutch, Swedish, Polish, Turkish, Thai, Vietnamese, Hindi, Greek, Hebrew
